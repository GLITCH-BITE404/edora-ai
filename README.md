# Edora AI

Edora AI is a web-based homework and learning assistant built to help students work smarter, not harder.  
The project focuses on clean UX, fast responses, and support for multiple languages and file-based inputs.

**Live app:**  
https://edora-ai.lovable.app

**GitHub repository:**  
https://github.com/GLITCH-BITE404/edora-ai

---

## What is Edora AI?

Edora AI is designed as a lightweight alternative to heavy AI platforms.

It allows users to:
- Ask homework and study questions
- Upload and analyze files (PDF support in progress)
- Switch languages dynamically
- Get clear, structured explanations instead of long, messy answers

The goal is accuracy, clarity, and speed.

---

## Tech Stack

This project is built with a modern frontend stack:
- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Lovable AI platform
